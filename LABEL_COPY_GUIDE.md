# 라벨 복사 기능 사용 가이드

## 📋 개요

라벨 복사 기능은 현재 프레임의 라벨(들)을 지정한 범위의 프레임들에 복사하는 기능입니다.

---

## 🎯 J 키 vs UI 라벨 복사 비교

| 기능 | 단축키 | 범위 | 설명 |
|------|--------|------|------|
| **간단 복사** | **J 키** | 다음 1개 프레임 | 현재 라벨을 바로 다음 프레임에 복사 |
| **범위 복사** | **UI 버튼** | 여러 프레임 | 라벨(들)을 지정한 범위에 복사 |

---

## 🎨 UI 라벨 복사 기능

### 위치
- 상단 회색 박스 안
- 오른쪽 영역 (마스킹 복사 옆)

### 구성
```
┌────────────────────────────────────────┐
│ 라벨 복사: ○전체 ○선택 ○다중선택       │
│ [선택된 라벨: 0개]                      │
│ [다중선택모드: OFF]                     │
│ ○유지 ○삭제                            │
│ 시작: [100] 종료: [200] [실행]         │
└────────────────────────────────────────┘
```

---

## 📝 복사 모드 3가지

### 1️⃣ 전체 모드 (기본값)
- **용도**: 현재 프레임의 모든 라벨을 복사
- **사용 예시**: 동일한 장면이 계속될 때

**사용 방법:**
1. **전체** 라디오 버튼 선택
2. 시작 프레임 입력 (예: 100)
3. 종료 프레임 입력 (예: 200)
4. **실행** 버튼 클릭
5. 현재 프레임의 모든 라벨이 100~200 프레임에 복사됨

### 2️⃣ 선택 모드
- **용도**: 현재 선택된 1개 라벨만 복사
- **사용 예시**: 특정 객체만 추적할 때

**사용 방법:**
1. **Tab 키**로 복사할 라벨 선택 (빨간색 테두리)
2. **선택** 라디오 버튼 선택
3. 시작/종료 프레임 입력
4. **실행** 버튼 클릭
5. 선택된 라벨만 복사됨

### 3️⃣ 다중선택 모드 ⭐ (이게 다중 복사 저장 기능!)
- **용도**: 여러 개의 라벨을 선택해서 복사
- **사용 예시**: 일부 객체들만 복사하고 싶을 때

**사용 방법:**
1. **"다중선택모드: OFF"** 버튼 클릭 → ON으로 변경
2. 복사할 라벨들을 마우스로 클릭 (노란색 점선으로 표시)
3. **다중선택** 라디오 버튼 선택
4. 시작/종료 프레임 입력
5. **실행** 버튼 클릭
6. 선택된 여러 라벨들이 모두 복사됨

## 🔧 기존 라벨 처리 옵션

복사 대상 프레임에 이미 라벨이 있을 때:

### ○ 유지 (기본값)
- 기존 라벨 유지
- 새로운 라벨 추가
- **결과**: 기존 라벨 + 복사된 라벨

### ○ 삭제
- 기존 라벨 모두 삭제
- 복사된 라벨만 남김
- **결과**: 복사된 라벨만

---

## 💡 사용 예시

### 예시 1: 전체 라벨 복사
```
상황: 프레임 1에 person, car, dog 라벨이 있음
목표: 프레임 10~50에 모두 복사

1. 프레임 1로 이동
2. "전체" 라디오 버튼 선택
3. 시작: 10, 종료: 50 입력
4. "실행" 클릭

결과: 프레임 10~50에 모두 person, car, dog 라벨 생성
```

### 예시 2: 특정 라벨만 복사
```
상황: 프레임 1에 person, car, dog가 있지만 person만 복사하고 싶음
목표: person 라벨만 프레임 10~50에 복사

1. Tab 키로 person 라벨 선택
2. "선택" 라디오 버튼 선택
3. 시작: 10, 종료: 50 입력
4. "실행" 클릭

결과: 프레임 10~50에 person 라벨만 생성
```

### 예시 3: 여러 라벨 선택해서 복사 ⭐
```
상황: person(3개), car(2개), dog(1개)가 있는데
      person 2개와 car 1개만 복사하고 싶음
목표: 선택한 라벨들만 복사

1. "다중선택모드: OFF" 버튼 클릭 → ON
2. 복사할 person 2개 클릭 (노란색 점선)
3. 복사할 car 1개 클릭 (노란색 점선)
4. "선택된 라벨: 3개" 확인
5. "다중선택" 라디오 버튼 선택
6. 시작: 10, 종료: 50 입력
7. "실행" 클릭

결과: 선택한 3개 라벨만 프레임 10~50에 복사
```

## 🎹 단축키 정리

| 키 | 기능 |
|----|------|
| **J** | 현재 라벨을 다음 프레임에 간단 복사 |
| **Tab** | 다음 라벨 선택 (선택 모드용) |
| **마우스 클릭** | 다중 선택 (다중선택 모드 ON일 때) |

---

## ⚠️ 주의사항

### 1. 다중선택 모드 활성화 필수
- **다중선택** 라디오 버튼을 선택하려면
- **먼저 다중선택 모드를 ON**으로 해야 함
- 그래야 라벨들을 클릭해서 선택 가능

### 2. 선택 확인
- **"선택된 라벨: X개"** 표시로 몇 개 선택되었는지 확인
- 노란색 점선 테두리로 시각적 확인

### 3. 프레임 범위
- 시작 프레임 ≤ 종료 프레임
- 1 ≤ 프레임 번호 ≤ 전체 이미지 개수

### 4. "유지" vs "삭제"
- **유지**: 기존 라벨 + 새 라벨 (누적)
- **삭제**: 새 라벨만 (교체)

## 🔄 워크플로우

### 빠른 복사 (J 키)
```
현재 라벨 → J 키 → 다음 1개 프레임에 복사
```

### 범위 복사 - 전체
```
모든 라벨 확인 → 전체 선택 → 범위 입력 → 실행
```

### 범위 복사 - 선택
```
Tab으로 라벨 선택 → 선택 모드 → 범위 입력 → 실행
```

### 범위 복사 - 다중선택 ⭐
```
다중선택 모드 ON → 라벨들 클릭 → 다중선택 모드 → 범위 입력 → 실행
```

## 🐛 문제 해결

### "선택된 라벨이 없습니다" 오류
- **선택 모드**: Tab 키로 라벨을 먼저 선택하세요
- **다중선택 모드**: 라벨들을 클릭해서 선택하세요

### 다중선택이 안 돼요
- "다중선택모드: OFF" 버튼을 클릭해서 ON으로 바꾸세요
- 버튼이 파란색이 되어야 합니다

### 복사가 안 돼요
- 시작/종료 프레임 번호가 올바른지 확인
- 라벨이 제대로 선택되었는지 확인
- 콘솔 출력을 확인하세요
