# 다중 라벨 선택 기능 사용 가이드

## 📋 개요

다중 라벨 선택 기능을 사용하면 여러 라벨을 한꺼번에 선택하여 일괄 작업을 할 수 있습니다.

---

## 🎯 사용 방법

### 1️⃣ 다중 선택 모드 활성화

**방법 1: UI 버튼 사용**
- **라벨 복사 영역**(상단)의 **"다중선택모드: OFF"** 버튼 클릭
- 버튼이 **"다중선택모드: ON"**으로 바뀌고 배경색이 **파란색**으로 변경됨
- 위치: 상단의 마스킹 복사/라벨 복사 영역 (회색 박스 안)

### 2️⃣ 라벨 선택하기

다중 선택 모드가 ON 상태일 때:
1. **마우스로 라벨을 클릭**하면 선택/해제됨
2. 선택된 라벨은 **노란색 점선 테두리**로 표시됨
3. 여러 라벨을 클릭하여 다중 선택 가능

**시각적 표시:**
- 🔴 **빨간색 실선**: 현재 선택된 라벨 (단일 선택)
- 🟡 **노란색 점선**: 다중 선택된 라벨들
- ⚪ **클래스별 색상**: 일반 라벨 (선택 안 됨)

### 3️⃣ 선택 정보 확인

하단에 선택된 라벨 정보가 표시됩니다:
```
선택된 라벨: 3개 (person, car)
```

### 4️⃣ 다중 선택된 라벨로 작업하기

#### 📋 라벨 복사 기능 사용
1. **"라벨 복사"** 버튼 클릭 또는 단축키 사용
2. 복사 다이얼로그에서 **"다중선택"** 라디오 버튼 선택
3. 복사할 프레임 범위 입력 (시작~종료)
4. **"실행"** 버튼 클릭
5. 선택된 모든 라벨이 지정한 범위에 복사됨

**사용 가능한 작업:**
- ✅ 라벨 복사 (범위 지정 가능)
- ✅ 일괄 삭제
- ✅ 클래스 변경

### 5️⃣ 다중 선택 모드 해제

- **"다중선택모드: ON"** 버튼을 다시 클릭
- 버튼이 **"다중선택모드: OFF"**로 변경됨
- 모든 다중 선택이 자동으로 해제됨

---

## 🔧 추가 기능: 다중 클래스 선택

특정 클래스들만 일괄 작업하고 싶을 때 사용합니다.

### 사용 방법:

1. **"다중 클래스"** 버튼 클릭
2. 팝업 창에서 작업할 클래스들 체크
3. 확인 버튼 클릭
4. 선택한 클래스들에 대해 일괄 작업 수행

---

## 💡 사용 예시

### 예시 1: 여러 라벨을 다른 프레임에 복사

```
1. "다중선택모드: OFF" 버튼 클릭 → ON으로 변경
2. 복사할 라벨들을 클릭 (노란색 점선으로 표시됨)
3. "라벨 복사" 버튼 클릭
4. "다중선택" 라디오 버튼 선택
5. 시작 프레임: 100, 종료 프레임: 200 입력
6. "실행" 클릭
7. 선택된 모든 라벨이 100~200 프레임에 복사됨
```

### 예시 2: 특정 라벨들만 삭제

```
1. 다중선택모드 ON
2. 삭제할 라벨들 클릭 (노란색 점선)
3. X 키 또는 삭제 버튼
4. 선택된 라벨들이 모두 삭제됨
```

### 예시 3: person 클래스만 모두 선택

```
1. "다중 클래스" 버튼 클릭
2. "person" 체크박스 선택
3. 확인
4. person 클래스 라벨들에 대해 일괄 작업 가능
```

## 🎹 단축키

현재는 버튼을 통해서만 다중 선택 모드를 활성화할 수 있습니다.

**다중 선택 모드에서:**
- **마우스 클릭**: 라벨 선택/해제
- **X 키**: 선택된 라벨들 삭제

---

## ⚠️ 주의사항

1. **다중 선택 모드 OFF 시 자동 해제**
   - 다중 선택 모드를 끄면 선택된 라벨들이 모두 해제됩니다
   - 작업 전에 선택 상태를 확인하세요

2. **프레임 이동 시 선택 유지**
   - 다중 선택은 현재 프레임에서만 유효합니다
   - 다른 프레임으로 이동하면 선택이 해제됩니다

3. **일반 모드와 차이**
   - 다중 선택 모드 OFF: 클릭 시 이전 선택이 해제되고 새로운 라벨 선택
   - 다중 선택 모드 ON: 클릭 시 라벨이 다중 선택 목록에 추가/제거

---

## 🐛 문제 해결

### 라벨이 선택되지 않아요
- **"다중선택모드: ON"** 상태인지 확인하세요
- 버튼 배경이 파란색이어야 합니다

### 선택된 라벨이 안 보여요
- 노란색 점선 테두리를 찾아보세요
- 하단의 "선택된 라벨: X개" 표시를 확인하세요

### 다중 선택이 자꾸 해제돼요
- 다중 선택 모드가 OFF로 바뀌지 않았는지 확인
- 프레임을 이동하지 않았는지 확인

---

## 📍 UI 위치

### 버튼 위치:
- **"다중 클래스"** 버튼: 상단 버튼 영역 (Back/Next 버튼 근처)
- **"다중선택모드: OFF/ON"** 버튼: 상단 라벨 복사 영역 (회색 박스 안)
- **"선택된 라벨: X개"** 표시: 다중선택모드 버튼 바로 왼쪽
