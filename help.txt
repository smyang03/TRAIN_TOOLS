=== GTGEN Tool 종합 도움말 ===

[프로그램 개요]
GTGEN Tool은 이미지 레이블링(Ground Truth Generation)을 위한 도구입니다.
객체 검출, 분류 작업을 위한 바운딩 박스 라벨을 생성하고 관리합니다.

==================================================
[기본 마우스 조작]
==================================================
- 좌클릭: 객체 선택
- 좌클릭 + 드래그: 객체 이동 또는 크기 조정
- Ctrl + 좌클릭: 새 객체 추가 모드 시작
- 우클릭: 다양한 컨텍스트 메뉴 (제외 영역 완성 등)
- 마우스 휠: 확대/축소 (Zoom In/Out)

==================================================
[키보드 단축키 - 기본 작업]
==================================================
**프레임 이동:**
- 좌 화살표(←) 또는 W: 이전 프레임
- 우 화살표(→) 또는 E: 다음 프레임
- U: 첫 페이지로 이동
- I: 마지막 페이지로 이동
- T: 왼쪽으로 스킵 (SkipFrame 간격)
- Y: 오른쪽으로 스킵 (SkipFrame 간격)

**객체 선택:**
- 위 화살표(↑): 다음 객체 선택
- 아래 화살표(↓): 이전 객체 선택
- Tab: 다음 객체 선택 (NextRect)
- Shift + Tab: 이전 객체 선택 (PreRect)
- C: 다음 사각형 (NextRect)

**객체 추가/삭제:**
- A: 객체 추가 모드 (Add)
- X: 선택한 객체 삭제 (Remove)
- D: 선택한 객체 삭제 (Delete)

**클래스 선택:**
- 1~9, 0: 설정된 클래스 번호 선택
- 각 클래스 버튼에 할당된 단축키 사용

==================================================
[키보드 단축키 - 객체 이동 및 크기 조정]
==================================================
**픽셀 단위 이동 (Shift + 화살표):**
- Shift + ←: 왼쪽으로 1픽셀 이동
- Shift + →: 오른쪽으로 1픽셀 이동
- Shift + ↑: 위로 1픽셀 이동
- Shift + ↓: 아래로 1픽셀 이동

**크기 조정 (Ctrl + 화살표):**
- Ctrl + ←: 폭 1픽셀 감소 (왼쪽)
- Ctrl + →: 폭 1픽셀 증가 (오른쪽)
- Ctrl + ↑: 높이 1픽셀 감소 (위)
- Ctrl + ↓: 높이 1픽셀 증가 (아래)

**크기 조정 (단축키):**
- F: 크기 줄이기 (ResizeDown)
- G: 크기 키우기 (ResizeUp)

[키보드 단축키 - 기타 기능]
==================================================
**확대/축소:**
- +: 확대 (Zoom In)
- -: 축소 (Zoom Out)

**표시 옵션:**
- `: 십자선 표시 (CrossLine)
- Shift + A: 클래스 표시 토글 (ViewClass)
- Shift + S: 선택한 객체만 표시 (OnlySelect)
- Shift + F: 박스만 표시 (OnlyBox)

**저장 및 로드:**
- S: 마스킹 저장 (MaskingSave)
- L: 마스킹 로드 (LoadMasking)
- B: 백업 복원 (ReloadBackup)
- ;: 현재 이미지 새로고침 (Reload)

**복사 및 복제:**
- Z: 복사 (CopyKey)
- J: 라벨 복사 (CopyLabeling)

**마스킹 기능:**
- M: 마스킹 모드 (Masking)
- N: 마우스 마스킹 (Mmasking)
- V: ROI 마스킹 (Masking_ROI)
- O: 크롭 모드 (Crop)

**자동 기능:**
- P: 자동 추적 (AutoTracking)
- K: 자동 라벨링 (AutoLabeling)
- R: 라벨링 리셋 (ResetLabeling)

**기타:**
- H: 도움말 표시 (Help)

[체크박스 옵션]
==================================================
- Show Size: 객체 크기 정보 표시 (픽셀 및 정규화 값)

- Mask→Del Labels: 마스킹 영역과 겹치는 라벨 삭제

==================================================
[고급 기능]
==================================================

**1. 제외 영역 기능 (폴리곤)**
전역 제외 영역을 설정하여 특정 영역의 라벨을 자동으로 삭제합니다.
모든 이미지에 적용됩니다.

사용 방법:
1. '제외영역' 버튼 클릭
2. '영역 추가' 클릭하여 폴리곤 그리기 모드 활성화
3. 캔버스에서 좌클릭으로 점 추가 (최소 3개)
4. 우클릭 또는 첫 점을 다시 클릭하여 폴리곤 완성
5. '기능 활성화' 체크
6. 다음 페이지부터 해당 영역의 라벨이 자동으로 삭제됨

관리:
- 삭제: 선택한 영역 삭제
- 활성화/비활성화: 영역별 토글
- 모두 삭제: 모든 제외 영역 삭제

표시:
- 주황색 테두리: 활성화된 제외 영역
- 회색 테두리: 비활성화된 제외 영역

**2. 클래스 자동 삭제 기능**
특정 클래스를 선택하면 다음 페이지부터 자동으로 삭제됩니다.

사용 방법:
1. '자동삭제' 버튼 클릭
2. 삭제할 클래스 체크박스 선택
3. '저장' 클릭
4. 다음 페이지부터 선택한 클래스의 라벨이 자동으로 삭제됨

팁:
- '모두 선택' / '모두 해제' 버튼으로 빠르게 설정
- 설정은 자동으로 저장되어 프로그램 재시작 시에도 유지됨

**3. 다중 클래스 선택**
여러 객체를 동시에 선택하여 일괄 작업할 수 있습니다.

사용 방법:
1. '다중 클래스' 버튼 클릭
2. 객체를 클릭하여 선택 (노란색 테두리)
3. 선택된 객체들을 일괄 삭제/수정

**4. 라벨 추적 모드**
프레임 간에 라벨을 자동으로 추적하여 연속된 프레임에서 동일한 객체를 찾습니다.

사용 방법:
1. 'Track Labels' 체크박스 활성화
2. 현재 프레임에서 추적할 객체 선택
3. 다음 프레임으로 이동하면 자동으로 유사한 위치의 객체를 찾아서 연결

==================================================
[마스킹 복사 기능]
==================================================
특정 프레임 범위의 마스킹 영역을 복사하거나 삭제할 수 있습니다.

**마스킹 복사:**
1. 시작 프레임 번호 입력
2. 종료 프레임 번호 입력
3. '실행' 버튼 클릭
4. 현재 프레임의 마스킹이 해당 범위에 복사됨

**범위 삭제:**
1. 시작 프레임 번호 입력
2. 종료 프레임 번호 입력
3. '실행' 버튼 클릭 (삭제 섹션)
4. 해당 범위의 라벨이 삭제됨

**페이지 이동:**
1. 이동할 페이지 번호 입력
2. 'Go' 버튼 클릭

==================================================
[라벨 복사 기능]
==================================================
선택한 라벨을 다른 프레임으로 복사할 수 있습니다.

**기본 복사:**
1. 복사할 라벨 선택
2. 대상 프레임 범위 입력 (시작~종료)
3. '실행' 버튼 클릭

**검색 및 복사:**
1. '검색' 버튼 클릭
2. 검색 조건 입력 (클래스, 크기 등)
3. 검색된 라벨을 선택하여 복사

==================================================
[설정 파일]
==================================================
**config.ini**
프로그램의 모든 단축키와 기본 설정이 저장됩니다.
- [Setting]: 기본 설정 (TPFP 분류, 스킵 프레임 등)
- [BasicKeySetting]: 기본 단축키 설정
- [SpecialKeySetting]: 특수 단축키 설정 (Shift, Ctrl 조합)

**클래스 설정 파일 (*.json)**
- 클래스 ID, 이름, 단축키, 색상 정보 저장
- '클래스 설정' 버튼으로 관리
- 최대 80개 클래스 지원

**자동 삭제 설정 (.auto_delete_classes.json)**
- 자동 삭제할 클래스 목록 저장
- '자동삭제' 버튼으로 관리

**제외 영역 설정 (.global_exclusion_zones.json)**
- 전역 제외 영역 폴리곤 정보 저장
- '제외영역' 버튼으로 관리

==================================================
[팁 & 트릭]
==================================================
1. 빠른 작업을 위해 단축키를 적극 활용하세요
2. 클래스 버튼은 우클릭으로 색상 변경 가능
3. 제외 영역은 모든 이미지에 적용되므로 한 번만 설정하면 됨
4. 자동 삭제 기능으로 불필요한 클래스를 일괄 제거
5. 라벨 추적 모드로 연속 프레임 작업 효율 향상
6. 슬라이더로 빠르게 이미지 이동 가능
7. Show Label List로 모든 라벨을 한눈에 확인

==================================================
[문제 해결]
==================================================
**프로그램이 멈춤:**
- Ctrl + C로 프로그램 종료 후 재시작
- original_backup 폴더 확인

**라벨이 저장되지 않음:**
- labels 폴더 쓰기 권한 확인
- 파일이 다른 프로그램에서 열려있는지 확인

**제외 영역이 동작하지 않음:**
- '기능 활성화' 체크 확인
- 제외 영역이 제대로 그려졌는지 확인 (최소 3개 점)

**자동 삭제가 동작하지 않음:**
- '자동삭제' 다이얼로그에서 클래스 선택 후 '저장' 클릭 확인
- 프로그램 재시작 후 설정이 로드되는지 확인
